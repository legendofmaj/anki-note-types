# About
Anki note types for word mining using TV. Based on Kaishi 1.5k's note type.
# Set up
In Anki click onto `Tools > manage note types`
create a new note type
    click `add` -> `basic` -> `ok`
add the following fields to it:
    1. `Word`
    2. `Word Meaning`
    3. `Sentence`
    4. `Sentence Meaning`
    5. `Picture`
    6. `Notes`
click `save`
open `cards`
    1. In `Front Template` paste `FrontTemplate.html`
    2. In `Back Template` paste `BackTemplate.html`
    3. In Styling paste `Styling.css`
click `save`
